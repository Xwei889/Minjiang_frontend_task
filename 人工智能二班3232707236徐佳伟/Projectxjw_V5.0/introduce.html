课程简介

本课程是软件工程专业的一门专业必修课。通过对Web前端三大主流开发技术学习、探究和实践，让学生理解和掌握 HTML5、CSS3、JavaScript等脚本语言的相关知识，通过项目实战和训练培养学生开发和设计Web网站或网页的基本操作技能和工作岗位适应能力。

通过本课程的学习，使学生理解和掌握 HTML（超文本标记语言）中各种常用标记的使用； CSS（层叠样式表）常用属性的使用，如何运用 div + CSS对网页进行布局； JavaScript脚本语言的基础知识，以及借助于 JavaScript语言的强大功能增强网页功能，实现用户交互事件等动态网页功能，为后续课程奠定学习基础。

课程核心内容

  • 🌐
    HTML5

    语义化标签、表单增强、多媒体支持、本地存储

  • 🎨
    CSS3

    Flex布局、Grid布局、动画过渡、响应式设计

  • ⚡
    JavaScript

    DOM操作、事件处理、异步编程、数据交互
